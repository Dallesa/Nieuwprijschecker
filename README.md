# 🏷️ Nieuwprijs Checker

**Nieuwprijs Checker** is een slimme browser-extensie die consumenten helpt besparen op tweedehands platformen zoals 2dehands.be en Marktplaats.nl door in real-time de huidige nieuwprijs bij partners (o.a. Bol.com) te tonen.

## 🚀 Over het project
Vaak is het prijsverschil tussen een tweedehands artikel en een nieuw exemplaar (met garantie en gratis verzending) verrassend klein. Deze tool scant automatisch de producttitel op de pagina en injecteert een subtiele prijs-alert als er een scherpe deal beschikbaar is bij een officiële webshop.

### Belangrijkste functionaliteiten:
- **Automatische Scan:** Werkt direct op 2dehands.be en Marktplaats.nl.
- **Real-time Vergelijking:** Haalt actuele prijzen op via officiële partner-API's.
- **Privacy-First:** Geen verzameling van persoonsgegevens of browsergeschiedenis.
- **Direct Besparen:** Schakel met één klik over naar een veilige aankoop met volledige garantie.

## 📁 Structuur
- `/` - De bronbestanden voor de officiële landingspagina ([nieuwprijschecker.nl](https://nieuwprijschecker.nl)).
- `/extension` - De broncode van de Chrome Web Store extensie (Manifest V3).

## 🛠️ Installatie (Development Mode)
1. Clone deze repository.
2. Open Chrome en ga naar `chrome://extensions/`.
3. Zet 'Ontwikkelaarsmodus' aan (rechtsboven).
4. Klik op 'Uitgepakt laden' en selecteer de map `/extension`.
